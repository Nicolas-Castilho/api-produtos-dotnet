# API de Produtos

## Descrição
API desenvolvida em ASP.NET Core para gerenciamento de produtos.

## Funcionalidades
- Criar produto
- Listar produtos
- Atualizar produto
- Deletar produto

## Tecnologias
- ASP.NET Core
- Entity Framework Core
- MySQL

## Como rodar
1. Clonar o projeto
2. Configurar connection string
3. Rodar aplicação
